Nova Aetas v4.0 - Reuploaded version - open beta
---------------------------------------

NOTE: this is open beta. You have two choices:
wait one week after September 1st for final non-savegame compatible release 
or play open beta now.


INSTALL INSTRUCTIONS:
1. Install all to ...\Warband\., not to the modules folder.
2. Accompanied with enb shaders addon, remove it if you want 
by deleting enbseries.ini.


ENBseries
This is a Shaders addon that causes the game to have a 
cinematic bloom. This is added to get the same shaders effect 
as in Bannerlord. It is said to not put a heavy burden on 
your system or any performance.
